# Car Diagnostic App

An IoT-based Android application for vehicle diagnostics and monitoring using Kotlin, Jetpack Compose, Room Database, and OBD-II technology.

## Features

* Vehicle Diagnostic Scanning
* Fault Code Detection
* Service Booking Module
* Vehicle History Tracking
* Workshop Finder
* Live RPM Monitoring
* Room Database Integration
* Modern Jetpack Compose UI

## Technologies Used

* Kotlin
* Jetpack Compose
* Android Studio
* Room Database
* OBD-II
* SQLite
